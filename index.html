Hyperdrive is a P2P filesystem.

It's designed to help you share files quickly and safely, directly from your computer.

The cross-platform Hyperdrive daemon deals with all the storage and networking details for you. The daemon's CLI lets you create, share, and inspect Hyperdrives in your library, and its gRPC API provides full programmatic control over daemon-managed drives.

If you're on Mac or Linux, the daemon also offers FUSE support, meaning your drives will appear as normal folders on your computer -- moving files in and out of Hyperdrive is a breeze.

It has all the features of Hypercore, plus a few extras:

  • Version Controlled: Files are versioned by default, making it easy to see historical changes and prevent data loss.
  • Composable: Using our mount system, Hyperdrives can be nested within other Hyperdrives, enabling powerful multi-user collaboration tools.
  • On-Demand Downloading: By default, readers only download the portions of files they need, on demand. You can stream media from friends without jumping through hoops! Seeking is snappy and there's no buffering.
  • Fast Lookups: File metadata is stored in a distributed trie structure, meaning files can be located with minimal network lookups.

Hypercore Hyperdrive Hyperswarm

Hyperswarm is a DHT for the home.

Its Kademlia-backed discovery system comes packed with tricks for staying fast and reliable in diverse network environments.

Peers within home networks come and go, and home router NATs often make it challenging to connect to them. Hyperswarm contains a novel distributed holepunching algorithm, which can traverse the vast majority of home NATs. When peers disappear, Hyperswarm deploys many heuristics to keep the DHT healthy in the presence of these unreliable nodes.

And the whole thing's UDP-based, so it makes thrifty use of your system's resources.

A few highlights:

  • Distributed Holepunching: If Peer A can be holepunched by any Peer B, then Peer C can bootstrap a connection to Peer A through Peer B. Holepunching is transitive, and can navigate through most home routers.
  • Adaptive Announcing: Peers will only announce when they're considered stable, leading to a healthier DHT with fewer unconnectable nodes.
  • Fast Lookups: Looking up a key only takes a few hundred milliseconds on average.

Hypercore Hyperdrive Hyperswarm

Hypercore is an append-only log

It comes with a secure transport protocol, making it easy to build fast and scalable peer-to-peer applications. Think lightweight blockchain crossed with BitTorrent.

Hypercore currently serves as the foundation for a diverse range of P2P applications, including chat apps, filesystems, databases,and even a browser. And the whole thing's MIT-licensed, so you can use it as you like!

It's full of useful features:

  • Simple Sharing: You can discover and sync Hypercores using our Hyperswarm DHT using a single 32-byte key.
  • Secure: Hypercore replication streams are encrypted using the NOISE Protocol, and blocks are exchanged using signed Merkle trees. It is end-to-end secure and tamper-proof.
  • Fast and Lightweight: Hypercores can replicate at gigabit speeds, and each core has a tiny memory footprint.
  • Sparse: You don't need to sync the entire log in order to read a few blocks. The protocol supports downloading only the blocks you need on demand.
  • Modular and Standardized: Both the replication protocol and the storage format are standardized, meaning you can hook it up to a vast assortment of storage and networking tools.
